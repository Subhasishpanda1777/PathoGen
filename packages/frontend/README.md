# PathoGen Frontend

React.js frontend with Neumorphism + Glassmorphism design system.

## Features

- ✅ Modern Neumorphism + Glassmorphism UI design
- ✅ React.js with Vite
- ✅ React Router for navigation
- ✅ Axios for API calls
- ✅ GSAP animations
- ✅ Responsive design
- ✅ Backend integration

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```
VITE_API_URL=http://localhost:5000
```

3. Start development server:
```bash
npm run dev
```

The app will run on http://localhost:3000

## Build

```bash
npm run build
```

## Pages

- `/` - Home page
- `/dashboard` - Health dashboard
- `/login` - Login page
- `/register` - Registration page
- `/report` - Symptom reporting
- `/medicines` - Medicine search
- `/privacy` - Privacy policy
- `/terms` - Terms of service
- `/admin/reports` - Admin reports panel

## Design System

- **Neumorphism**: Soft shadows and embossed effects
- **Glassmorphism**: Frosted glass effects with backdrop blur
- **Colors**: Modern palette with high contrast
- **Typography**: Inter font family

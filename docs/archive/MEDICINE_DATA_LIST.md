For finding images for these medicines, search for:
1. **Brand Name** (e.g., "Crocin tablet", "Calpol 500")
2. **Generic Name + Form** (e.g., "Paracetamol 500mg tablet")
3. **Manufacturer + Brand** (e.g., "GSK Crocin", "Cipla Azithral")
4. **Packaging** (e.g., "Crocin 10 tablets strip")

Most medicines will have:
- Tablet/Capsule: Strip packaging images
- Syrup: Bottle images
- Cream: Tube images
- Inhaler: Inhaler device images
- Eye Drops: Bottle with dropper images
